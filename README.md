# 🎤 Sistema de Eventos, Palestrantes e Clientes - Modelagem de Banco de Dados

## 📖 Sobre o Projeto

Este projeto apresenta a modelagem de um **Sistema de Gerenciamento de Eventos**, desenvolvida utilizando a ferramenta **brModelo** com o objetivo de representar e organizar as informações relacionadas a eventos, palestrantes e clientes.

A modelagem foi criada para demonstrar como os dados podem ser estruturados em um banco de dados relacional, permitindo o controle de inscrições, palestras, participantes e organização de eventos de forma eficiente.

---

## 🎯 Objetivos

* Compreender os conceitos de modelagem de banco de dados.
* Desenvolver diagramas utilizando o brModelo.
* Identificar entidades, atributos e relacionamentos.
* Aplicar conceitos de cardinalidade e normalização.
* Representar processos de gerenciamento de eventos.
* Preparar a estrutura para futura implementação em um SGBD.

## 🎪 Contexto do Projeto

A modelagem foi desenvolvida para representar um sistema responsável pelo gerenciamento de eventos, contemplando informações relacionadas a:

* Eventos
* Palestrantes
* Clientes
* Participantes
* Inscrições
* Palestras
* Programação
* Certificados
* Categorias de Eventos

O objetivo é garantir a organização e o gerenciamento eficiente das informações necessárias para o planejamento e execução de eventos.
